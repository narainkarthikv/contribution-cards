# Contributing to Contribution Cards

Thanks for your interest in contributing to Contribution Cards. This project is community-driven and welcomes all levels of experience.

## Code of Conduct

Please read and follow the [Code of Conduct](./CODE_OF_CONDUCT.md).

## Ways to Contribute

- Report bugs or request features
- Improve documentation or examples
- Fix issues or add enhancements
- Refine UI/UX and accessibility
- Optimize performance and caching

## Getting Started

1. Fork the repository and create a new branch from the default branch.
2. Keep your changes focused and small when possible.
3. Test your changes locally before opening a PR.

## Development Setup

Prerequisites:

- Node.js 18.0.0 or higher
- npm 9.0.0 or higher

Install and run:

```bash
npm install
npm run dev
```

Quality checks:

```bash
npm run lint
npm run build
```

## Pull Request Guidelines

- Provide a clear summary of what changed and why.
- Link related issues when applicable.
- Include screenshots or short clips for UI changes.
- Ensure the app builds and linting passes.

## Commit Guidelines

Use clear, descriptive commit messages. Example:

```
feat: add repository filter badges
```

## Reporting Issues

When filing a bug, include:

- Steps to reproduce
- Expected vs actual behavior
- Screenshots or logs (if helpful)
- Environment details (OS, browser, Node version)
