# Security Policy

## Supported Versions

We support the latest release on the default branch. If you are using an older commit, please upgrade before reporting issues.

## Reporting a Vulnerability

If you believe you have found a security vulnerability:

1. Do not open a public issue with full details.
2. Contact the maintainer via GitHub: https://github.com/narainkarthikv
3. Provide steps to reproduce, impact, and any relevant logs.

We will acknowledge your report and work with you on a fix and disclosure timeline.
